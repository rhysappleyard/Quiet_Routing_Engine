# Quiet_Routing_Engine
An app for finding the quietest feasible route through a city or otherwise. For those who want to take the scenic route. 


Starting with Barcelona. 
Realised that the Eixample's grid is a trap for simple algorithms because all paths look equal in length.
Some roads may be more quiet than others for a range of reasons so I have changed from tracking the type-of-road data (with OSMNX) to real reported data on Barcelona from OpenData BCN.
